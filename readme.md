## Prerequisites
- Cmake, min version 3.14

## Build:
Should work for most general systems/compilers, but tested so far with windows/MSVC:  
- git clone https://github.com/mako542b/tetris
- cd tetris/build
- cmake ..
- cmake --build . --config Release
